# public/ — Static Assets

Files in this folder are served directly to the browser without being processed by the build system.

## Files

| File | Purpose |
|------|---------|
| `index.html` | The HTML shell that loads the React app — contains the root `<div>` the app mounts into |
| `manifest.json` | Web app manifest — defines the app name, icons, and theme for when it's installed as a PWA |
| `admin-patients-qa-checklist.md` | QA checklist for admin and patients features |
| `dashboard_loose_ends_checklist.md` | Checklist of remaining dashboard items to finish |
