# common/ — Shared Components

Components that are used across multiple pages throughout the app.

## Files

| File | Purpose |
|------|---------|
| `ClinicianTopBar.tsx` | The navigation bar at the top of every clinician page — shows the app logo, clinician name, Messages button with unread count, and Logout |
| `StatusBadge.tsx` | A small coloured pill that shows a patient's status (Stable/Warning/Critical) — converts AI risk levels into visual labels |
