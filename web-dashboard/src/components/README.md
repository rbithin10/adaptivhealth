# components/ — Reusable UI Building Blocks

This folder contains all the reusable visual pieces that pages put together to build their layouts. Components are organised into subfolders by type.

## Subfolders

| Folder | Purpose |
|--------|---------|
| `cards/` | Small display cards that show a single statistic or metric |
| `common/` | Shared components used across multiple pages (top navigation bar, status badges) |
| `patient/` | Components specific to viewing a patient's health data (vitals, alerts, risk, ML panels) |
