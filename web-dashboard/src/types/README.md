# types/ — Data Shape Definitions

This folder defines the structure of all data used in the app. These TypeScript interfaces describe what the backend server sends back, so the frontend code knows exactly what fields to expect.

## Files

| File | Purpose |
|------|---------|
| `index.ts` | All data type definitions — covers authentication, users, vital signs, risk assessments, recommendations, alerts, activities, ML features (anomaly detection, trend forecasts, explainability), messaging, medical profiles, consent management, and admin operations |
