# theme/ — Visual Styling

This folder defines the app's visual identity — colours, fonts, and Material-UI theme configuration.

## Files

| File | Purpose |
|------|---------|
| `colors.ts` | All colour values used in the app — brand blue, critical red, warning amber, stable green, neutral grays, and chart colours |
| `typography.ts` | Text style presets — page titles, section titles, body text, captions, big numbers, and more |
| `muiTheme.ts` | Material-UI theme configuration — applies the custom colours and fonts to all MUI components |
