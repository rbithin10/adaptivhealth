# assets/ — Static Resources

Images, data files, and ML models bundled with the app.

## Subfolders

| Folder | Contents |
|--------|----------|
| `data/` | JSON data files (e.g., `recipes.json` — heart-healthy recipe library) |
| `exercises/` | Exercise illustration images (PNG) — arm raises, cycling, swimming, yoga, etc. |
| `images/` | Screen background images (PNG) — login, home, health, profile, recovery, workout, etc. |
| `ml_models/` | On-device machine learning models (TFLite format) for cardiovascular risk prediction — deployed separately from app builds |
