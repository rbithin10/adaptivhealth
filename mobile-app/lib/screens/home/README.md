# screens/home/ — Home Tab Widget Sections

Modular widgets that make up the Home dashboard tab. Each file renders one section of the Home screen.

## Files

| File | Purpose |
|------|---------|
| `home_heart_rate_ring.dart` | The central heart rate circle — shows live BPM with a risk-coloured glow ring |
| `home_sparkline.dart` | Heart rate sparkline chart — line graph of today's readings from live or API data |
| `home_vitals_grid.dart` | Secondary vitals row — horizontally scrollable cards for SpO2, Blood Pressure, and HRV |
| `home_quick_actions.dart` | Four quick-action buttons — Start Workout, Recovery, Health, and AI Coach |
| `home_recommendation_card.dart` | "Recommended For You" section — AI-suggested activity with duration and HR zone |
| `home_recent_activity.dart` | Recent workout list — shows past exercise sessions with type, duration, and time ago |
| `home_rehab_card.dart` | Rehab programme card — shows active rehab progress (hidden when not in a programme) |
