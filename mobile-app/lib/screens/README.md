# screens/ — App Screens

Each file here is a full-page view that the user can navigate to. These are the main "pages" of the app.

## Screens

| File | What the user sees |
|------|--------------------|
| `login_screen.dart` | Email and password sign-in form |
| `register_screen.dart` | New account registration form |
| `onboarding_screen.dart` | First-time health profile setup wizard (age, conditions, medications, etc.) |
| `home_screen.dart` | Main dashboard with heart rate ring, vitals grid, quick actions, and AI recommendations |
| `health_screen.dart` | Detailed health data with trends, charts, and historical insights |
| `profile_screen.dart` | User profile, settings, theme toggle, and account management |
| `nutrition_screen.dart` | Daily meal logging and calorie/nutrient tracking |
| `recipe_library_screen.dart` | Browse and search heart-healthy recipes |
| `fitness_plans_screen.dart` | AI-recommended workout plans based on cardiovascular profile |
| `workout_screen.dart` | Active workout tracking with timers and heart rate zones |
| `recovery_screen.dart` | Post-workout recovery exercises and cooldown routines |
| `rehab_program_screen.dart` | Cardiac rehabilitation program progress and phases |
| `history_screen.dart` | Historical health data and activity log |
| `device_pairing_screen.dart` | Bluetooth device scanning and pairing for heart rate monitors |
| `doctor_messaging_screen.dart` | Secure messaging with the user's clinician |
| `notifications_screen.dart` | In-app notifications and health alerts |

## Subfolders

| Folder | Purpose |
|--------|---------|
| `home/` | Smaller widget pieces that make up the home dashboard (heart rate ring, vitals grid, quick actions, etc.) |
