# theme/ — Visual Styling

Defines the app's look and feel — colors, fonts, and theme configurations. All screens reference these files so the design stays consistent.

## Files

| File | Purpose |
|------|---------|
| `colors.dart` | The complete color palette — brand colors, status colors (safe/warning/critical), background shades, and dark mode variants |
| `typography.dart` | Text styles for headings, body text, labels, metric values, and buttons |
| `theme.dart` | Combines colors and typography into Flutter's theme system so all widgets automatically use the right styles |
